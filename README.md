## Install

```bash
$ npm install cymbaline
```

Or if you are using **yarn**:

```bash
$ yarn add cymbaline
```

## Run tests

Clone this repository to your local machine:

```bash
$ git clone https://github.com/ghanizadev/cymbaline.git
$ cd cymbaline
```

Install its dependencies:

```bash
$ npm install
```

Run tests:

```bash
$ npm run test
```
